## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/cydexx/chatgpt-dnd-character-generator
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the server  

~~~bash  
npm run dev
~~~
